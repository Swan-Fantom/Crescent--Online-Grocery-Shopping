# Crescent--Online-Grocery-Shopping
Crescent, is an online platform for shopping groceries. It is a full stack project created using JavaScript, MongoDB, ExpressJS, NodeJS, AJAX, CSS, Bootstrap and EJS. Feel free to install the codes and check it out

The Dependencies used in this project are:-
